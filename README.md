**Программа-помощник для режима Снеговик/Турбовик в STALCRAFT:X**

## Для чего она?
- Вы вводите свое поле с игры для анализа программой. Она показывает какие ячейки пустые и какие потенциально золотые. Она не предсказывает, судит только по тому что вы ввели (Внимательным программа не понадобится)

## Начать использовать:
- Качайте последний релиз. Открываете html файл в папке. Он откроется в браузере - так и должно быть;
- В случае возникновений багов и прочего - будут выходить новые релизы.

## Как пользоваться?
- Нажатие на ячейку меняет её тип: Неизвестная,обычная,золотая;
- Режим "Золотые контуры" позволит вам повесить золотые рамки на ячейки;
- "Новая игра" сбросит все что вы сделали на поле;
- Анализ происходит автоматически.
